Term Substitution

BitwuzlaTerm bitwuzla_substitute_term(Bitwuzla *bitwuzla, BitwuzlaTerm term, size_t map_size, BitwuzlaTerm map_keys[], BitwuzlaTerm map_values[])

Substitute a set of keys with their corresponding values in the given term.

Parameters

  • bitwuzla – The Bitwuzla instance.

  • term – The term in which the keys are to be substituted.

  • map_size – The size of the substitution map.

  • map_keys – The keys.

  • map_values – The mapped values.

Returns

The resulting term from this substitution.

void bitwuzla_substitute_terms(Bitwuzla *bitwuzla, size_t terms_size, BitwuzlaTerm terms[], size_t map_size, BitwuzlaTerm map_keys[], BitwuzlaTerm map_values[])

Substitute a set of keys with their corresponding values in the set of given terms.

The terms in terms are replaced with the terms resulting from this substitutions.

Parameters

  • bitwuzla – The Bitwuzla instance.

  • terms_size – The size of the set of terms.

  • terms – The terms in which the keys are to be substituted.

  • map_size – The size of the substitution map.

  • map_keys – The keys.

  • map_values – The mapped values.
