Options

A bitwuzla::Bitwuzla instance is created from a configuration options bitwuzla::Options instance.

Note

This options instance must be configured before creating the Bitwuzla instance. After the Bitwuzla instance is created, configuration options are fixed and cannot be changed.

Every option can be configured via void bitwuzla::Options::set(const std::string &lng, const std::string &value). Additionally, Boolean and numeric options are configured via void bitwuzla::Options::set(Option option, uint64_t value), and options with modes are configured via void bitwuzla::Options::set(Option option, const std::string& value). The option kind is defined via bitwuzla::Option. A comprehensive list of all configurable options is available here.

More information on and an example for how to configure and query configuration options is here.

Note

Some options are labeled as “expert” options. Use with caution.
